##  Closure



클로저는 코드의 블록이다.

함수 = 이름이 있는 클로저 

클로저를 익명함수라고 부르지 말것!!

일급 시민 / 일급 객체

변수 상수 등으로 저장 가능, 전달인자로 전달이 가능

클로저는 함수의 전달인자로 많이 사용된다.

### 클로저 형태

```swift
{ (매개변수 목록) -> 반환타입 in
	실행 코드
}
```



### 클로저 사용

```swift
var sum: (Int, Int) -> Int = { (a: Int, b: Int) -> Int in
                             return a+ b
                             }
```
